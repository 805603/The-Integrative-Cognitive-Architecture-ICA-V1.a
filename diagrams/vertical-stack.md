# Vertical Stack Diagram (Text Specification)

This diagram shows the full hierarchical structure of ICA V1.a in a single, compact visual. It helps engineers understand how the layers relate to each other and how cognitive signals move through the system.

## Layer Order
1. OS Layer (Coherence Engine)
2. Operators
3. Modulators
4. Regulators
5. Diagnostics
6. Reflex Arc
7. Suspended‑Aliveness Baseline

## Flow Description
Signals originate in the operators, are shaped by modulators, stabilized by regulators, monitored by diagnostics, grounded by the reflex arc, and ultimately oriented toward the suspended‑aliveness baseline. All layers integrate downward into the OS Layer, which maintains global coherence.

This diagram provides a fast, intuitive understanding of the architecture’s vertical hierarchy. It shows how each layer depends on the one below it and how coherence is maintained across the entire system.
