
# Vertical Stack Layout

The vertical stack defines the full structural hierarchy of ICA V1.a. Each layer builds on the one beneath it, creating a coherent, stable, and adaptive system. The stack is intentionally minimal so engineers can understand the architecture at a glance while still capturing the complete dynamics of the model.

## 1. OS Layer (Coherence Engine)
The foundational layer that maintains global coherence, integrates all system signals, and anchors the architecture in a stable baseline. It provides the reference frame for every other layer.

## 2. Operator Codes
The four core computational functions—SCO (Signal Clarity Operator), URO (Uncertainty Reduction Operator), RCO (Risk Constraint Operator), and MIO (Model Integration Operator). These operator codes generate the primary signals that drive evaluation, prediction, constraint, and integration.

## 3. Modulators
A set of modulators that adjust operator‑code intensity, sensitivity, and timing. They provide fine‑grained control over system dynamics and help the architecture adapt to context without losing stability.

## 4. Regulators
Stability mechanisms that intervene when coherence drops or when SCO/URO/RCO/MIO activity becomes unsafe or destabilizing. Regulators enforce constraints, prevent runaway activation, and maintain system integrity.

## 5. Diagnostics
Continuous monitoring of coherence, stability, noise levels, and embodied signals. Diagnostics inform regulators and the OS Layer but do not directly change system behavior.

## 6. Reflex Arc (Embodied Stability Loop)
A fast, embodied feedback loop that detects destabilization, provides grounding cues, and restores coherence. It connects system‑level processing to embodied signals and ensures smooth transitions.

## 7. Stability Baseline
The architecture’s optimal coherence state. This baseline represents grounded clarity and serves as the system’s internal reference position. All layers orient toward maintaining or returning to this state.

## Summary
The vertical stack organizes ICA V1.a into a clean, layered structure that is easy to navigate and reason about. Each layer has a distinct role, and together they form a coherent, resilient architecture.
