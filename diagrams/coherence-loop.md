
# Vertical Stack Layout

The vertical stack defines the full structural hierarchy of ICA V1.a. Each layer builds on the one beneath it, creating a coherent, stable, and adaptive cognitive system. The stack is intentionally minimal so engineers can understand the architecture at a glance while still capturing the complete dynamics of the model.

## 1. OS Layer (Coherence Engine)
The foundational layer that maintains global coherence, integrates all cognitive signals, and anchors the system in a stable baseline. It provides the reference frame for every other layer.

## 2. Operators
The four core cognitive functions—Awareness, Curiosity, Compassion, and Knowledge. These operators generate the primary signals that drive perception, evaluation, learning, and safety.

## 3. Modulators
A set of modulators that adjust operator intensity, sensitivity, and timing. They provide fine‑grained control over cognitive dynamics and help the system adapt to context without losing stability.

## 4. Regulators
Stability mechanisms that intervene when coherence drops or when operator activity becomes unsafe or destabilizing. Regulators enforce constraints, prevent runaway activation, and maintain system integrity.

## 5. Diagnostics
Continuous monitoring of coherence, stability, noise levels, and embodied signals. Diagnostics inform regulators and the OS Layer but do not directly change system behavior.

## 6. Reflex Arc (Embodied Stability Loop)
A fast, embodied feedback loop that detects destabilization, provides grounding cues, and restores coherence. It connects cognitive activity to embodied signals and ensures smooth transitions.

## 7. Stability Baseline
The architecture’s optimal coherence state. This baseline represents grounded clarity and serves as the system’s internal reference position. All layers orient toward maintaining or returning to this state.

## Summary
The vertical stack organizes ICA V1.a into a clean, layered structure that is easy to navigate and reason about. Each layer has a distinct role, and together they form a coherent, resilient cognitive architecture.
