# Integrative Cognitive Architecture (ICA) — V1.a
Canonical specification of the Integrative Cognitive Architecture (ICA) V1.a by Sean Meehan.

ICA is a unified cognitive framework that integrates curiosity, coherence, adaptive control, and intrinsic stability into a single architecture. It provides a structured model for understanding and designing intelligent systems that remain stable, curious, and self-correcting across environments.

## Purpose
This repository serves as the permanent, public, and canonical archive of ICA V1.a. It includes the formal definitions, operator structure, controller layers, modifier set, and the complete architectural rationale.
